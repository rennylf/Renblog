柳絮之城
----

跟老同学们吃完晚饭，从吾悦广场里出来，大概八点多。Z 君与我相伴，步行穿过洞山小街。我在一台ATM 机上取了现金，约了车准备回酒店。那时我们分开站在路边，身后是所学校，依稀可以看出红钻白墙，但在假期的夜晚，是人去楼空的状态。樟树叶摇曳作响，恰似阵阵翻书声，分不清楚拨动的是清风还是时间，我脑海中浮现一句歌词 「Time is still turning the page.」

转眼毕业快三年，J 君的婚礼，对我们 705 的室友而言是个重逢的契机。我提前一天结束工作，早上从坪山赶到深圳北，随后乘坐高铁一路北上，穿过广东、江西，在安徽中部的小城淮南落脚，最后当出租车载着我，行驶在淮南大道上时，日落的最后一丝余晖，也正撒在路边的树丛中。久在深圳，感慨其他城市都相对开阔，没有密集的钢筋水泥墙。等到抵达酒店，办完手续，Z 君已经等候多时。他给我开门：“你怎么一点都没有变！” 我同样寒暄回去，语言中藏着某种关乎时间的退让。

![](../images/nvr2vdkEf6ULoSBibA640wx_fmtpngfromappmsg.png)

直到第二天，仪式开始前，新郎 J 君终于有段空闲，来酒桌前和我们说话，讨论这些年，彼此的一些变化。我们用手机录了一段，还用另一台手机展示着，未能到场的室友照片，达成了寝室的再次团聚。仪式在主持人的引导，按照流程进行，我和 Z 君分两侧守着门，直到新娘和 J 君 在舞台的中央相逢，一时间他们都流下了泪水，那似乎隐藏着很多不为外人道的故事。在感动之余，唯有祝福他们。

婚礼结束，J 君一定要送我去车站，犹如岸边马蹄。我站在安徽理工大学对面的商业街，一处树荫下，等他开车过来。作为南方人，我第一次见柳絮因风起的模样，如同一首悠扬的乐曲。伸手想要抚摸，它们却如同音符亲吻了你的耳边，又转身飘远。整个车程只有十来分钟，到了淮南南，我走上进站的平台，J 君从车上下来，再跟我说一些话，关于感谢，关于有机会再聚。现代人送别总是短暂的，无论是三分钟限停、还是他人催促的喇叭，还是我即将出发的下一程列车，都让我们匆匆做别。

古镇写生
----

我的下一站是黄山，老朋友 X 君已到了那里。坐在列车上的当儿，我偏执地相信，也许它开往的并非将来，而是开往过去。当 X 君提及想去徽州时，我悄然把这当成对那次缺席之旅的弥补，即使多年里我从未想起：2016 年，我在建城上学，因为某些原因，我没有和同学们按原计划去徽州写生。

![](../images/EnHZeVkZh14dELSW0A640wx_fmtpngfromappmsg.png)

我们去了呈坎、西递、宏村三个古镇，刻舟求剑般的找寻我想象中的写生队伍。他们或许四五成群，带着粗布折叠凳上坐在树荫下，黑色铁制画架晒的发烫，偶尔轻风吹动那些夹在浅棕色的画板的白纸。他们一定有双未谙世事的双眼，好像从未考虑过明天的烦恼，不像早已浑浊的颜料桶，染成五颜六色画毛刷。但那里只有熙熙攘攘的人群，与我擦肩而过。

![](../images/aDriaLWgb2mXXoMPg640wx_fmtjpegfromappmsg.jpeg)

X 君自诩「风光佬」，已经有摄影师的风范，比如他带上了好几台摄像设备（虽然忘记带三脚架成了遗憾），比如他总会找到最佳拍照机位，并事先踩点。又比如他会在天未亮，游客未至的时候，出门拍片。这些我不得不佩服。往往我是等他早出归来，再一起出去走走。

![](../images/aDAWEiapTlJf1ribw640wx_fmtjpegfromappmsg.jpeg)

这三处的古镇都是徽派建筑，屋外以马头墙为主要特色，白墙青瓦，又有阶梯状层叠的直线条造型，给人朴素、雅致的观感，整个村落房屋建筑密度高，高低错落，以山野、日落为背景，以水池做点缀，置身其中，仿佛走在古画里。“如果下雨那就好了。” X 君期待拍出烟雨古镇的质感。在我进入房子内部时，我也总觉得洒落进来的，应该是雨水，而非阳光。也许在百年以前，屋子的主人正歪在躺椅上，看着雨帘，听着雨水滴落在中庭的蓄水池，敲打着荷叶，发出令人昏昏欲睡的声音。

![](../images/NMN1AvbknqHnId09SQ640wx_fmtpngfromappmsg.png)

失落烟渚
----

千岛湖是我们旅行的最后一站，它坐落在杭州市淳安县，是一处为修建水库而围成的人工湖泊。在湖水灌入后，此处原本的崇山峻岭被淹入，化成了大小千来个岛屿。抵达千岛湖那个晚上，我们实在已经无法忍受当地清淡的饮食，所以决心找一家川菜或湘菜，好在我们最后得偿所愿，在某家川菜店填满了口腹之欲。

在我们大快朵颐的时候，隔壁桌的两个女生和老板攀谈了起来。老板是一位看起来花甲之年的老妪。她正在讲述着辣酱是如何在重庆老家制作，然后运送过来，在别的地方绝对吃不到这个味道。转而谈起对小店命运的担忧，说在这里的客人不算多，又开始宽慰自己，说自己开店只是为了找一份事事情做，其实她并不需要考虑生计问题。我被这些故事吸引，侧耳朵听她继续讲，她是如何辗转来到此处，她却被伙计叫走了，估计有活要忙。我多少有点失望，和 X 君打车回去了。

![](../images/m5Xbtx8tnqRiafq9g640wx_fmtjpegfromappmsg.jpeg)

大概湖边、海边等地方的旅行项目都逃不过骑行这个项目。希望这是最后一次在旅途中骑行，我心里想。然后跟着 X 君租了一辆电瓶车，开始了环湖骑行。这确实是一片很大的湖泊，很难想象是人力干预而成。岛屿上站满了葱郁的树木，在靠近水面出漏出赤裸的沙色腰身。因为身体不适，我们只骑了一上午，下午在酒店休息了过去。晚上，我们心心念念的雨，在措不及防的时间到来，我们正从麦当劳吃完出来，去还电瓶车的路上。幸运地是，它还算温柔，并没有让我们浑身湿透。

搭上去往车站的大巴时，还不到早上八点。车外小雨淅沥，让窗外的景色朦胧起来。我们再次路过湖边，小岛们雾气缭绕。我想起某位故交，他的签名是「移舟泊烟渚，日暮客愁新」我被烟渚（弥漫雾气的水中沙洲）的意象吸引，所以与他结识，也许他也旅经过一片沙洲，他当时会想一些什么？我无法得到他的答案，唯有书写我的答案。
